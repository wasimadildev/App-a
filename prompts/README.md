## How to Use

### Midjourney
- Copy the `### Midjourney Prompt` block for each image
- Paste directly into Discord with `/imagine`
- Keep the `--ar`, `--v`, `--s`, and `--style raw` parameters

### DALL-E 3
- Copy the `### DALL-E 3 Prompt` block
- Paste into ChatGPT or DALL-E interface
- Use the aspect ratio info to set the correct crop

### Stable Diffusion
- Copy the `### Stable Diffusion Prompt` block
- Use the Positive/Negative fields and settings provided
- Recommended checkpoint: Realistic Vision or Juggernaut XL

## Export Instructions

After generating, convert all images to WebP at 85% quality (use Squoosh.app or ImageMagick). Save to `/public/images/` with the filenames listed in each prompt file.
